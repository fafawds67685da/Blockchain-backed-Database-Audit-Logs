# Blockchain Audit System - React Frontend

## Installation

```bash
cd d:\Blockchain-backed-Database-Audit-Logs\frontend-react
npm install
```

## Running the App

### Terminal 1: Backend
```bash
cd d:\Blockchain-backed-Database-Audit-Logs
uvicorn backend.main:app --reload
```

### Terminal 2: Frontend
```bash
cd d:\Blockchain-backed-Database-Audit-Logs\frontend-react
npm start
```

The app will open at **http://localhost:3000**

## Features

✅ Modern React UI with Tailwind CSS
✅ Real-time dashboard with charts
✅ Employee management (CRUD operations)
✅ Blockchain integrity verification
✅ Tampering simulation & detection
✅ Email notifications
✅ Transaction history with Etherscan links
✅ Gas fee tracking
✅ Analytics & insights
✅ CSV & PDF export

- React 18
